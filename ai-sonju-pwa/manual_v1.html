AI 손주란 무엇인가요?

AI 손주는 인공지능이 손주처럼 매일 어르신의 건강을 챙겨드리는 서비스입니다.
복잡한 기기 조작 없이, 평소 사진 찍어 보내듯 간단하게 사용할 수 있어요.

📷
식사 사진 영양 분석
사진 한 장을 보내면 AI가 3초 안에 영양소를 분석하고 점수를 드려요
💊
약 복용 알림
설정한 시간이 되면 자동으로 알림이 오고 복용 여부를 확인해요
📱
자녀 자동 알림
식사·약 복용 정보가 자녀에게 자동으로 전달돼요
💬
따뜻한 대화
AI 손주와 안부 인사, 건강 조언 등 편하게 대화할 수 있어요
💡 이런 분께 딱 맞아요! 혼자 사시는 어르신, 자녀가 멀리 사는 경우, 매일 약을 드셔야 하는 분, 건강한 식사를 챙기고 싶으신 분
2

처음 시작하기 (설정 방법)

처음 앱을 열면 설정 화면이 나타나요. 아래 순서대로 입력해 주세요.
5~10분이면 충분해요!

1
⚙️ 오른쪽 위 톱니바퀴 버튼을 눌러요
화면 오른쪽 위에 있는 ⚙️ 버튼을 눌러 설정 화면을 여세요.
2
👴 "기본" 탭 — 어르신 정보 입력
성함과 나이를 입력해 주세요.
입력하면 AI 손주가 이름으로 불러드려요. 예) "복순 어르신, 안녕하세요!"
3
👨‍👩‍👧 자녀 연락처 추가
자녀 이름과 전화번호를 입력하고 "추가" 버튼을 누르세요.
여러 명 등록 가능해요. 식사 사진을 보낼 때마다 자녀에게 자동 알림이 가요.
4
💊 "약 관리" 탭 — 복용 약 등록
약 이름, 용량, 복용 시간, 알람 시간을 입력하고 "추가"를 누르세요.
예) 혈압약 · 1정 · 아침 식후 · 오전 8:00
5
📱 "알림" 탭 — 알림 항목 선택
식사 알림, 약 복용 알림, 미복용 경고 등 원하는 항목만 켜두세요.
처음에는 모두 켜두시는 걸 추천드려요.
6
💾 "설정 저장" 버튼 클릭
맨 아래 주황색 "설정 저장" 버튼을 눌러야 저장돼요.
이 버튼을 꼭 눌러 주세요!
✅ 설정 완료 확인 방법 화면 위쪽에 "김복순님 · 자녀 2명 · 약 3종" 처럼 정보가 표시되면 설정이 완료된 거예요!

약 복용 관리

매일 드셔야 하는 약을 등록하면 알람이 울리고, 복용 여부를 간단하게 체크할 수 있어요.

💊
설정 필요
약 등록하기 (처음 한 번만)
⚙️ 설정 → 💊 약 관리 탭에서 등록하세요.
입력 항목:
• 약 이름 — 예) 혈압약, 당뇨약, 비타민
• 용량 — 예) 1정, 2알
• 복용 시간 — 아침 식후 / 점심 식후 / 저녁 식후 / 취침 전
• 알람 시간 — 예) 오전 8:00
• 메모 (선택) — 예) 식사 30분 후 복용
⏰
자동이에요
알람 시간이 되면 자동 알림이 와요
설정한 시간에 채팅 창에 메시지가 오고, 알림 팝업도 떠요.
"⏰ 혈압약 드실 시간이에요! (아침 식후 08:00) 1정 챙겨드세요 😊"

쉬워요

자녀 알림 설정

자녀가 바빠서 연락하기 어렵더라도 AI 손주가 대신 소식을 전해드려요.
어떤 알림을 보낼지 항목별로 선택할 수 있어요.

🍚
식사 알림
"오늘 어머니가 아침 식사를 맛있게 하셨습니다 😊 건강 점수 88점!"
💊
약 복용 완료 알림
"어머니가 혈압약을 복용하셨습니다 💊"
⚠️
약 미복용 경고 알림
"어머니가 혈압약을 아직 복용하지 않으셨습니다. 확인 부탁드립니다 ⚠️"
📱 알림 ON/OFF 설정 위치 ⚙️ 설정 → 📱 알림 탭에서 항목별로 켜고 끌 수 있어요. 원치 않는 알림은 끄면 됩니다.

완전 무료

API 키란?
API 키 발급 방법 (Gemini 추천)
API 키는 AI 서비스를 사용하기 위한 비밀번호 같은 거예요.
1. aistudio.google.com 접속
2. 구글 계정으로 로그인
3. "Create API key" 버튼 클릭
4. 생성된 키(AIza로 시작) 복사, 모델선택:Gemini 2.5 Flash(최신,유료)
5. AI 손주 ⚙️ 설정 → 🤖 AI 탭 → Gemini 선택 → 키 붙여넣기 → 저장
🤖 API 키 없어도 사용 가능해요! API 키를 입력하지 않아도 "데모 모드"로 기본 기능을 체험할 수 있어요. 실제 AI 분석을 원하실 때 입력하세요.

홈 화면에 앱으로 추가하기

스마트폰 홈 화면에 추가하면 앱 아이콘을 탭하는 것만으로 바로 열려요.
주소를 외울 필요가 없어요!

🤖
안드로이드 폰
1. 크롬 브라우저로 열기
2. 주소창 옆 ⋮ 메뉴 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭 → 완료! ✅
🍎
아이폰
1. 사파리로 열기
2. 아래 공유 버튼 □↑ 탭
3. "홈 화면에 추가" 탭
4. 오른쪽 위 "추가" → 완료! ✅
📲 설치 후에는 이렇게 사용하세요 홈 화면의 🤖 AI 손주 아이콘을 탭하면 앱처럼 전체 화면으로 열려요. 인터넷 주소를 다시 입력할 필요가 없어요!

자주 묻는 질문

사진을 보내도 아무 반응이 없어요
API 키가 입력되지 않은 경우 데모 모드로 동작합니다. ⚙️ 설정 → 🤖 AI 탭에서 API 키를 입력하고 저장해 보세요. 키가 없으면 샘플 사진 선택 후 데모 답변을 확인해 보세요.
약 알람이 오지 않아요
앱이 완전히 닫혀 있으면 앱 내 알람은 울리지 않아요. 알람 시간에 앱을 열어두시거나, 스마트폰의 알림 설정에서 이 앱의 알림이 허용되어 있는지 확인해 주세요. 또한 설정 저장을 누르셨는지 확인해 주세요.
자녀에게 실제로 문자가 가나요?
현재 버전은 앱 안에서 "자녀 알림이 발송되었습니다"라고 보여주는 시뮬레이션 기능입니다. 실제 문자·카카오 발송은 카카오톡 채널 연동 버전에서 지원됩니다. 기관 담당자에게 문의해 주세요.
설정이 저장되지 않아요
설정 화면 맨 아래의 주황색 "💾 설정 저장" 버튼을 반드시 눌러야 저장돼요. 버튼을 누른 후 "설정이 저장되었습니다" 메시지가 나오면 정상 저장된 거예요.
API 키가 뭔가요? 꼭 필요한가요?
API 키는 AI 서비스를 사용하기 위한 인증번호 같은 거예요. 없어도 기본 기능(데모 모드)은 사용 가능하지만, 실제 AI가 사진을 분석하려면 필요해요. Gemini(구글)는 구글 계정만 있으면 무료로 발급받을 수 있어요.
카카오톡 없이도 사용할 수 있나요?
네! PWA 버전은 카카오톡 없이 웹 브라우저로 바로 사용할 수 있어요. 기관 담당자에게 앱 링크를 받아서 열면 됩니다. 홈 화면에 추가하면 앱처럼 사용할 수 있어요.
개인 정보가 안전한가요?
입력한 이름, 자녀 연락처, 약 정보는 본인 기기에만 저장되고 외부 서버로 전송되지 않아요. 식사 사진은 AI 분석 후 즉시 삭제됩니다. AI API 사업자(Anthropic, OpenAI, Google)의 개인정보 처리 방침을 따릅니다.
여러 기관에서 같이 사용할 수 있나요?
네! 기관마다 별도로 설정하면 여러 기관에서 독립적으로 사용할 수 있어요. 각 기관이 원하는 AI(Claude, ChatGPT, Gemini)를 선택해서 사용할 수 있습니다.
